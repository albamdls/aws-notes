2Ô∏è‚É£ Section 2: AWS services and service category overview

üü† AWS foundational services

  • Built on: Regions, AZs, PoPs.
  • Deliver: networking, compute, storage, databases.
  • On-demand utility ‚Üí available in seconds, pay-as-you-go.

üü† AWS categories of services

  • AWS has 23 service categories.
  • Exam focus categories:
    • Compute
    • Storage
    • Database
    • Networking & Content Delivery
    • Security, Identity & Compliance
    • Management & Governance
    • Cost Management

üü† Storage service category

  • Amazon S3 ‚Üí object storage, scalable, secure, used for web apps, backup, IoT, analytics.
  • Amazon EBS ‚Üí block storage for EC2, high performance, used for DBs, analytics, containers.
  • Amazon EFS ‚Üí managed NFS file system, scales to petabytes, grows/shrinks automatically.
  • Amazon S3 Glacier ‚Üí low-cost archival storage, 11 9s durability, compliance.

üü† Compute service category

  • Amazon EC2 ‚Üí resizable compute capacity (VMs).
  • EC2 Auto Scaling ‚Üí adjust instances based on conditions.
  • Amazon ECS ‚Üí container orchestration for Docker.
  • Amazon ECR ‚Üí container registry for Docker images.
  • Elastic Beanstalk ‚Üí simple web app deployment.
  • AWS Lambda ‚Üí run code without servers (serverless), pay per execution.
  • Amazon EKS ‚Üí managed Kubernetes service.
  • AWS Fargate ‚Üí run containers without managing servers/clusters.

üü† Database service category

  • Amazon RDS ‚Üí managed relational DB, automates provisioning, patching, backups.
  • Amazon Aurora ‚Üí MySQL/PostgreSQL-compatible, 3‚Äì5x faster than standard DBs.
  • Amazon Redshift ‚Üí petabyte-scale data warehouse, queries on Redshift + S3.
  • Amazon DynamoDB ‚Üí key-value/document NoSQL DB, millisecond latency, built-in security.

üü† Networking and content delivery service category

  • Amazon VPC ‚Üí isolated networks in the AWS Cloud.
  • Elastic Load Balancing (ELB) ‚Üí distributes traffic across multiple resources.
  • Amazon CloudFront ‚Üí global CDN, low latency content delivery.
  • AWS Transit Gateway ‚Üí connect VPCs and on-prem networks to one gateway.
  • Amazon Route 53 ‚Üí scalable DNS service, routes domains to apps.
  • AWS Direct Connect ‚Üí dedicated private connection to AWS.
  • AWS VPN ‚Üí secure tunnels to AWS.

üü† Security, identity and compliance service category

  • IAM ‚Üí manage users, groups, permissions.
  • AWS Organizations ‚Üí multi-account management with SCPs.
  • Amazon Cognito ‚Üí user authentication and access for apps.
  • AWS Artifact ‚Üí compliance/security reports and agreements.
  • AWS KMS ‚Üí encryption key management.
  • AWS Shield ‚Üí managed DDoS protection.

üü† AWS cost management service category

  • AWS Cost and Usage Report ‚Üí detailed billing and usage.
  • AWS Budgets ‚Üí alerts when forecasted/actual usage exceeds budget.
  • AWS Cost Explorer ‚Üí visualize and analyze costs.

üü† Management and governance service category

  • AWS Management Console ‚Üí web-based management interface.
  • AWS Config ‚Üí track resource inventory and changes.
  • Amazon CloudWatch ‚Üí monitor resources and apps.
  • AWS Auto Scaling ‚Üí scale multiple resources dynamically.
  • AWS CLI ‚Üí unified command-line management tool.
  • AWS Trusted Advisor ‚Üí optimization insights (cost, performance, security).
  • AWS Well-Architected Tool ‚Üí review workloads against best practices.
  • AWS CloudTrail ‚Üí track API calls and user activity.
